# Динамический шутер с генерацией уровней
Представляет из себя шутер с генерацией уровней на основе блоков, простеньким ИИ врагов.

# Что выполнено:
- Стрельба по RayCast
- ИИ врагов
- Замедление времени (при нажатии на колёсико мыши)
- Постепенное усложнение уровней

# Управление
- WASD - движение
- Зажатый Shift - бег
- Пробел - прыжок
- ЛКМ - стрельба
- Нажатие на колёсико - вкл/выкл замедления времени
- R - перезарядка
